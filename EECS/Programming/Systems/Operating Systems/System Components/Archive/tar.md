# tar

## archive + gzip
```bash
tar czf ../dyn.tgz ./
```
## archive + bzip2
```bash
tar cjf ../dyn.tbz ./
```
## archive + xz
```bash
tar cJf ../dyn.txz ./
```
## archive + lzip
```bash
tar --lzip -cf ../dyn.lzip ./
```
## archive + lzma
```bash
tar --lzma -cf ../dyn.lzma ./
```
## archive + lzop
```bash
tar --lzop -cf ../dyn.lzop ./
```
## archive + compress
```bash
tar --compress -cf ../dyn.compress ./
```
